This folder contains the necessary files and folders that are used in a Marxan Server installation. 

empty_project       A template for a project which is copied to a new project folder with all the neccessary files
Start project       A simple project that is copied over to a users folder when a new user is registered
server.dat          Server-wide configuration variables, e.g. ENABLE_GUEST_USER which allows guest users to access the server and CORS_DOMAINS which specifies a comma separated list of domains that are allowed access to the server
user.dat            A template for a user which is copied to the users folder when a new user is registered
